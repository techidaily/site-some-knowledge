M2TS is a Sony high definition video file type. **M2TS** files are raw AVCHD videos recorded using Sony's camcorders, such as the HDR-SR1 and HDR-SR5 models. Panasonic, Canon and other brands of AVCHD camcorders also record in M2TS format. There are some media players for [playing M2TS on Mac](https://tools.techidaily.com/macxdvd/products/) and Windows, like Mplayer, Windows Media Player or other media players if there is a codec for DVD playing installed. The file type M2TS plays well only on high-profile computers, it is usually can not be read by portable players (PS4 is an exception).

### Transcode M2TS to MP4 for playback on portable devices

If you want to play M2TS videos on portable devices like iPad, iPod, iPhone, Apple TV, PSP, PS4 and so on, you have to convert M2TS files to playable formats at first. Here I recommend you convert M2TS to MP4 format for two reasons. First, MP4 is widely used on the most popular devices. More and more portable devices are support this file form. Second, MP4 has a much smaller file size and meanwhile remain the high video quality.
