## Step-by-Step Guide to Setting Up System Protection and Creating a Restore Point on Windows 10

To do a System Restore in Windows 10, you need to create a restore point first. Just follow steps below to create a restore point in Windows 10.  
  
1\. Open[Control Panel](https://tools.techidaily.com/drivereasy/download/).  
  
2\. View by**Small icons**and click on**System**.  
  
![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874ae59063e3.jpg)   
  
 3\. Click**System Protection** in left pane.

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874af17efd1d.jpg) 

  
4\. Under**Protection Settings**, ensure the**Protection**for the drive that you want to create a restore point is On (Note the drive must have Windows installed on.).  
  
![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874afbed9ce9.jpg)   
  
 If not, select the drive and click**Configure** to turn it on. 

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874b097d5efe.jpg) 

  
5\. Click**Create**button.  
  
![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874b0f127364.jpg)   
  
 6\. Type a description in the box, which will help you identify the restore point. (In my case, I type “byte”.) Then click**Create** button. After that, Windows will start creating the restore point.

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874b12d0ce25.png)   
  
 If you want, you can check whether the create point is created successfully:  
  
 Open **System Properties** again and click**System Restore** .

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874b6488c49c.jpg)   
  
 In my case, the “byte” restore point has been created successfully.

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874b63273174.jpg)
